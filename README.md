Flux-Toggler (Flux worflow for Alfred)
======
  Control Flux directly from alfred.

# Commands
| keyword | Command |
| ------ | ------ |
| fd1 | Disable for 1 Hour |
| fm | Movie Mode |
| fsun | Disable until sunrise |
| fcurr | Disable for current app |
| fdm | Darkroom Mode |
| fpref | Preferences |
| fquit | Quit f.lux |

### Require
  * [Alfred](https://www.alfredapp.com)(with Powerpack)
